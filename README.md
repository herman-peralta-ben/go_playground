# Go playground 

Tiny Go examples

* Install Go
https://go.dev/doc/install

* Create a simple Go project with:

```bash
mkdir simple-grpc && cd simple-grpc
go mod init example.com/simple-grpc
```

* Add dependencies with `go get <lib>`, e.g. `go get local/shared`.
